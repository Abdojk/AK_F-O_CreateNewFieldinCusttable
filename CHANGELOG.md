# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [1.0.0] - 2026-02-12

### Added
- Initial project structure for D365 F&O Customer field extensions
- AK_Nasa field (String 60) added to CustTable via table extension
- Form extension to display AK_Nasa on the General tab of the Customer form
- AKNasa Extended Data Type (EDT)
- CustCustomerV3Entity data entity extension for import/export support
- Model descriptor (AKCustTableExtensions)
- CLAUDE.MD permanent AI memory file
- PowerShell scripts for database sync, model export, and memory file management
- GitHub Actions workflow for CLAUDE.MD validation
- Technical documentation (design, deployment, decision log)
